# golang-recipes



## Uninstallation
```sh
sudo apt-get remove golang-go
sudo apt-get remove --auto-remove golang-go
```

## Installation
```sh
tar -C /usr/local -xzf go1.14.2.linux-amd64.tar.gz
wget https://dl.google.com/go/go1.14.2.linux-amd64.tar.gz
tar -C /usr/local -xzf go1.14.2.linux-amd64.tar.gz
```
